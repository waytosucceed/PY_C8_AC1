Returning Values

The return statement ends the function and tells it to send a value back to the part of the program that called it. It allows a function to produce a result that can be used elsewhere in the program.


Imagine you order groceries from an online grocery store. You expect the store to deliver the order to you.
If they just prepare it but never deliver it, you don't get the order you paid for.

The return statement is like the delivery of the order. The function does some work (prepares the order) and then returns the result (delivers the order) back to you.
